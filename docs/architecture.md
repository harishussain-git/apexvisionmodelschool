# Architecture

## App Router Sitemap

```text
/
|-- layout.js                 (Root layout applied to all routes)
|-- page.js                   (Home page)
`-- dev/
    `-- page.jsx              (Development page)
```

## Layout Hierarchy

```text
RootLayout (app/layout.js)
`-- children (route page content)
    |-- Home (app/page.js)
    `-- Dev Page (app/dev/page.jsx)
```

## Page Layout Breakdown

### `/` (Home)

```text
div
`-- h1  "Site Under Development"
```

### `/dev`

```text
div
|-- HeroSection
|   `-- section
|       |-- h1  "Hero Section"
|       `-- p   "Update this block with your hero content."
`-- Classroom
    `-- section
        |-- h2  "Classroom"
        `-- p   "Update this block with your classroom content."
```

## Notes

- Global styles are loaded from `app/globals.css` in the root layout.
- Global metadata is currently defined in `app/layout.js`:
  - title: `Create Next App`
  - description: `Generated by create next app`
